Test Instructions

To test this page:

  1. Start your Django server: python manage.py runserver
  2. Open this page: http://localhost:8000/static/media_test.html
  3. Check if all images load correctly
  4. Red borders indicate failed image loads

Expected Results

✓ Working: Images should load without red borders
✗ Not Working: Images with red borders indicate URL issues

Troubleshooting

If media images don't load:

  • Check that MEDIA_URL and MEDIA_ROOT are set in settings.py
  • Verify that media serving is configured in urls.py
  • Ensure the media directory exists and contains images
  • Check that Django is running in DEBUG mode
